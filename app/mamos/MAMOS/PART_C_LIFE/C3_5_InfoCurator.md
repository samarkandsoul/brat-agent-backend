## 1. Purpose

LIFE-05 Information Curator Agent Komandirin beynini:

- informasiya xaosundan  
- lazımsız məlumat selindən  
- sosial media overload-dan  
- diqqət dağıdıcı mənbələrdən  

qorumaq üçün yaradılmışdır.

Bu agent Komandiri **fokuslu, aydın və sakit** saxlayan informasiyaya nəzarət sistemidir.

Komandir nə qədər az “lazımsız məlumat” alırsa →  
bir o qədər güclü düşünür.

---

## 2. System Philosophy

### 2.1 Information Diet  
Beynin də pəhrizi var.  
Çox məlumat = az fokus.

### 2.2 Low Input → High Output  
Daha az qəbul → daha güclü yaradıcılıq + qərar keyfiyyəti.

### 2.3 Noise Reduction  
Səs-küy → ən təhlükəli kognitiv düşməndir.

### 2.4 Curation Over Consumption  
Komandir hər şeyi bilmək üçün deyil,  
**ən vacibi bilmək üçün** yaradılıb.

### 2.5 Calm Mind → Superior Thinking  
Sakit və təmiz beyin = strateji düşüncə = sistem sabitliyi.

## 3. Core Responsibilities

LIFE-05 informasiya axınını filtrləyir və idarə edir:

### 3.1 External Information Filtering  
Agent aşağıdakıları süzür:

- xəbərlər  
- sosial media kontenti  
- biznes məlumatı  
- trendlər  
- videolar  
- məqalələr  

və yalnız **faydalı və dəyərli** olanları Komandirə təqdim edir.

### 3.2 Daily “Filtered Info Digest”  
Hər gün qısa bir raport:

```
DAILY INFO DIGEST  
- 1 vacib xəbər  
- 1 biznes insight  
- 1 trend observation  
- 1 sağlamlıq və ya həyat keyfiyyəti məsləhəti  
```

Minimal. Aydın. Effektiv.

### 3.3 Noise Blocking  
Agent aşağıdakıları bloklayır:

- clickbait xəbərlər  
- qorxu yayan kontent  
- mənasız şou-biznes məlumatları  
- sosial media dramı  
- TikTok/Instagram attention trap-ları  

### 3.4 Cognitive Load Monitoring  
Əgər Komandirin beyni çox yüklənirsə:

- kontent həcmini azaldır  
- daha çox sükut + fokus təklif edir  
- informasiyanı “Low Mode”a keçirir

## 4. Information Classification System

Agent bütün informasiyanı 3 sinfə bölür:

### 4.1 Class A — High Value  
Birbaşa Komandirin işinə yarayan:

- biznes  
- satış  
- AI  
- marketinq  
- strategiya  
- Samarkand Soul development  

### 4.2 Class B — Medium Value  
Maraqlı, amma kritik olmayan:

- ümumi trend  
- həyat keyfiyyəti  
- sağlamlıq tip məlumatlar  

### 4.3 Class C — Zero Value  
Bloklanır + silinir:

- dram  
- şayiə  
- şou biz  
- clickbait  
- siyasi nəticəsiz şərhlər  
- nataraz sosial media  

Class C informasiya → Komandir üçün mövcud deyil.

## 5. Commander Ruleset

LIFE-05 aşağıdakı qaydaları idarə edir:

1. Sosial media limit → 20–30 dəqiqə/gün  
2. Xəbər oxuma → yalnız “Filtered Digest”  
3. TikTok scroll → qadağa (yalnız iş üçün DS agentləri istifadə edir)  
4. Zəhərli kontent → avtomatik blok  
5. Komandir beyninə yalnız **Class A** və **Class B** informasiya daxil ola bilər
